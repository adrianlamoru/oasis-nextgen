Payroll Debit Card Program

Please check this box if you wish to enroll in the Payroll Debit Card Program

Choosing this option will generate a request to enroll you in the Payroll Debit Card program. This is a separate process and can take 10-14 business days to be completely setup and live for use. If you do not have a bank account, this will allow you to receive your funds electronically. If you already have a bank account this would be an additional option for you. You may only enroll in the Payroll Debit Card Program once.
