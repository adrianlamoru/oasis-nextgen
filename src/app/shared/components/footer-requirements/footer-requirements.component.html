Oasis Website Requirements

Recommended Browsers for Windows Users*

  • Chrome 60.x
  • Firefox 55.x
  • Microsoft Internet Explorer 10.x
  • Microsoft Internet Explorer 11.x
  • Microsoft Edge

Recommended Browsers for Apple MacIntosh Users*

  • Chrome 60.x
  • Firefox 55.x
  • Safari 8.x

There are known limitations in pdf support depending on your OS version.

*Javascript is required.

Browser requires support for 2048 bit certificate with recommended cipher capability of 256 or higher.

Connection

This service recommends using a DSL, Cable Modem or faster connection for optimum experience. Please note, the speed at which you actually connect to the Internet may vary causing different performance results.
